Los elementos HTML se componen de una etiqueta de inicio, una de cierre y el contenido dentro de dichas etiquetas.
Los atributos modifican un elemento o etiqueta. la mayoria de los atributos tienen un valor; el valor modifica el atributo. Los atributos siempre estan especificados dentro de la etiqueta de apertura.
Con el atributo "width" podemos modificar la anchura de un elemento HTML, ya sea con pixeles "px" o con porcentaje "%" (hr width="500px")↑ .
El atributo "align" es utilizado para alinear elementos Html
Ya sea texto o elementos insertados.
La etiqueta "img" es usada para insertar una imagen y no lleva cierre. La URL de la imagen se define con el atributo "src".
Para que el explorador pueda desplegar la imagen, debemos colocar entre comillas la ubicacion de la imagen como valor del atributo src.
En caso de que la imagen no se pueda mostrar, el atributo alt especifica un texto que describe a la imagen en palabras.
Para definir el tamaño de una imagen, se usan los atributos "width"(ancho) y "heigth"(alto), y su valor se puede especificar en pixeles o porcentaje.
